## Description
Turbo Builder creates builds for multiple platforms in one click. It works together with Turbo Switch to offer an incredible fast build pipeline.

Turbo Builder is a great time-saving tool for your multi-platform projects.

Please read the "TurboBuilder-doc.pdf" and "TurboBuilder-api.pdf" for more details.

 


## Upgrade to new version
Follow these steps to upgrade the version of "Turbo Builder PRO":

1. Update "Turbo Builder PRO" to the latest version from the "Unity AssetStore"
2. Delete the "Assets/Plugins/crosstales/TurboBuilder" folder from the Project-view
3. Import the latest version from the "Unity AssetStore"
